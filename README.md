# Simulador da Copa do Mundo 2026

Aplicação desenvolvida em React Native utilizando Expo.

## Funcionalidades

- Seleção de classificados da fase de grupos
- Simulação das oitavas de final
- Simulação das quartas de final
- Simulação das semifinais
- Simulação da final
- Escolha manual ou aleatória dos vencedores
- Definição do campeão
- Reinício completo da simulação

## Tecnologias Utilizadas

- React Native
- Expo
- TypeScript
- JavaScript
